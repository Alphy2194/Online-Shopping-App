PROFOUND GRID



A responsive grid system for fixed and fluid layouts. Built in SCSS. It gives you flexibilty and full control



Responsive

Make the grid fluid or fixed and add permedia layouts as needed. Change the column or gutter widths and remove or add columns as needed

Semantic

The grid keeps your markup semantic and works without any "grid_X" classes. Unless you want to use them in which they are included

Precise

Profound grid uses negative margins to calculate columns. Unlike with the other grid systems, fluid layouts will look exactly the same in every browser.
